Instruções de preparação

Para preparar corretamente seu tabuleiro para o jogo, basta digitar a letra da embarcação desejada e colocá-la no tabuleiro usando o click do mouse.

  • Use a tecla R para o Cruzador
  • Use a tecla S para o Submarino
  • Use a tecla E para o Hidro Avião
  • Use a tecla W para o Encouraçado
  • Use a tecla Q para o Porta Aviões
  • Use a tecla U para girar a embarcação no sentido horário
  • Use a tecla I para girar a embarcação no sentido anti-horário
